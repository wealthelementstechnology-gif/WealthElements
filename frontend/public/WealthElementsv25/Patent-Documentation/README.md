### System Architecture:
- **Input:** 22-dimensional feature vector
- **ML Model:** Neural network (4,175 parameters)
  - Batch Normalization layer
  - Hidden Layer 1: 64 neurons, ReLU, L2 (0.001), Dropout (0.2)
  - Hidden Layer 2: 32 neurons, ReLU, L2 (0.001), Dropout (0.2)
  - Hidden Layer 3: 16 neurons, ReLU, L2 (0.0005)
  - Output Layer: 3 neurons, Sigmoid
- **Training:** Adam optimizer, MSE loss, Early stopping (patience=15)
- **Deployment:** TensorFlow.js (browser-based)
- **Performance:**
  - Prediction latency: <100ms per goal
