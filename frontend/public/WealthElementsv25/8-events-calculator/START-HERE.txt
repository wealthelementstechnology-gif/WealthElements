⚠️ COMMON ISSUES
══════════════════════════════════════════════════════════════════════

Issue: "TensorFlow.js failed to load" (ML not working)
Fix: Check internet connection. Files now try 3 different CDNs
     automatically. If all fail, system uses rule-based fallback.
     Refresh the page - it should work after 1-2 tries.

Issue: "Need 50+ training samples"
Fix: Use ml-data-manager.html → Generate Test Data tab

Issue: "QuotaExceededError" (storage full)
Fix: Use ml-data-manager.html → Cleanup tab → Keep 100 samples

Issue: Training button disabled
Fix: Check ml-data-manager.html → See what's missing

Issue: Wrong counts showing
Fix: ml-data-manager.html → Check Data tab → Click "Fix Data"
