Feature 1: 8 Events Calculator

Comprehensive Life-Stage Planning Tool

What It Does:

Calculates exactly how much you need to invest monthly to achieve all 8 major financial goals simultaneously.

How It's Different:

  • Integrated Approach: Most calculators handle one goal at a time. We optimize across all 8 events simultaneously.
  • Inflation-Adjusted: Automatically factors in 6-10% annual inflation for each goal category.
  • Surplus-to-SIP Optimizer: Input your income and expenses; we calculate your investable surplus and optimal allocation.
  • Visual Progress Tracking: See all 8 goals on one dashboard with completion percentages.

User Benefit:

"Know exactly where every rupee should go - no guesswork, no Excel sheets, just a clear plan."

Calculation Methodology

Step 1: Future Value Calculation

FV = PV × (1 + inflation_rate)^years

Example:
- Child's education today: ₹10,00,000
- Inflation rate: 10% (education-specific)
- Years left: 15 years
- FV = ₹10,00,000 × (1.10)^15 = ₹41,77,248

Step 2: Required SIP Calculation

SIP = FV × r / [(1 + r)^n - 1]

Where:
- FV = Future Value needed
- r = monthly return rate (annual_return / 12)
- n = number of months

Example:
- FV needed: ₹41,77,248
- Expected return: 12% p.a. (1% monthly)
- Years: 15 (180 months)
- SIP = ₹41,77,248 × 0.01 / [(1.01)^180 - 1]
- SIP = ₹7,091/month

Step 3: Lumpsum Impact

Lumpsum Future Value = PV × (1 + r)^n
Adjusted SIP = Original SIP - (Lumpsum FV impact)

Example:
- Existing lumpsum: ₹1,00,000
- Returns: 12% p.a.
- Years: 15
- Lumpsum FV = ₹1,00,000 × (1.12)^15 = ₹5,47,357
- Adjusted target = ₹41,77,248 - ₹5,47,357 = ₹36,29,891
- New SIP = ₹6,164/month (saving ₹927/month!)
