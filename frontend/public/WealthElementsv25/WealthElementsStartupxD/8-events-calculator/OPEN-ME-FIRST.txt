❌ IF TESTS FAIL
══════════════════════════════════════════════════════════════════════

Problem: "TensorFlow.js failed to load"
Fix: Check internet connection (loads from CDN)

Problem: "Scripts not loading"
Fix: Make sure all files are in same folder:
  - outcome-tracker.js
  - ml-features.js
  - ml-model.js
  - step6-ml-integration.js
  - step6.js
  - step6.html

Problem: "optimizePlan not found"
Fix: Hard refresh browser (Ctrl+Shift+R or Cmd+Shift+R)

Problem: Console shows errors
Fix: Check error message, try different browser
